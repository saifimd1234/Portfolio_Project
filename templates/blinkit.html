Blinkit Dynamic Dashboard

Interactive Excel Dashboard for Sales Analytics

About the Project

This Excel dashboard analyzes Blinkit's sales data using pivot tables, slicers, and dynamic visualizations. Features include real-time KPI tracking, interactive filters, and brand-aligned design.

Interactive Dashboard

Key Features

  • Data Standardization: Cleaned raw data using Excel’s table formatting
  • Dynamic KPIs: Total sales, average ratings, and item count metrics
  • Interactive Filters: Slicers for outlet size, location tier, and item type
  • Visual Analytics: Donut charts for fat content, line charts for sales trends

Future Enhancements

  • Live API data integration for real-time updates
  • Advanced chart types like heatmaps and scatter plots
  • Mobile-responsive dashboard version
  • User-customizable KPI panels
