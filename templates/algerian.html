Exploratory Data Analysis (EDA) üîç

  1. Data Cleaning: Detect and handle missing values and outliers.
  2. Data Exploration: Generate summary statistics and visualize data with histograms, box plots, and heatmaps.
  3. Outlier Detection: Use methods like Z-score and box plots to pinpoint anomalies.
  4. Insights: Differentiate between categorical and numerical data to uncover hidden patterns.

Feature Engineering (FE) üîß

  1. Feature Selection: Identify impactful features such as FFMC, DMC, temperature, and humidity.
  2. Scaling & Normalization: Apply StandardScaler or MinMaxScaler for improved model accuracy.
  3. Encoding: Transform categorical variables using label or one-hot encoding.
  4. Derived Features: Create new features (e.g., dryness index) by combining temperature and humidity.
  5. Dimensionality Reduction (Optional): Use techniques like PCA to optimize training speed.
